BATTLE SYSTEMS FLOWCHART
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

Turn Order:
  1. Sort units by Speed (highest first)
  2. Each unit gets one turn
  3. Can Move + Act per turn
  4. Round ends when all units acted
  5. Update status effects
  6. Check victory conditions
  7. Start new round

Player Turn:
  1. Display battle grid
  2. Show movement range
  3. Player selects action:
     - Move to valid position
     - Attack enemy in range
     - Wait (end turn)
     - View stats
  4. Execute action
  5. Next unit's turn

AI Turn:
  1. Check if enemies in melee range
  2. If yes: Attack weakest enemy
  3. If no: Move toward nearest enemy
  4. Attack again if now in range
  5. End turn

Combat Resolution:
  1. Calculate hit chance
  2. Roll for hit/miss
  3. If hit:
     - Calculate base damage
     - Apply critical (if rolled)
     - Apply elemental modifier
     - Apply terrain defense
     - Apply height advantage
     - Add random variance
  4. Deal damage to target
  5. Apply status effects (20% chance)
  6. Check if target defeated
